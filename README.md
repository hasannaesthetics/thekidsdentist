## 🎨 Color Customization

Want to adjust colors? All colors are CSS variables in `styles.css`:

```css
:root {
    --primary: #FF6B9D;        /* Pink */
    --secondary: #4ECDC4;      /* Teal */
    --accent: #FFD93D;         /* Yellow */
    --text: #2C3E50;           /* Dark gray */
    --text-light: #64748B;     /* Light gray */
}
```

Change these values to rebrand instantly!
